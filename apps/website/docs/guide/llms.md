# LLMs

This project follows the [llms.txt](https://llmstxt.org) proposed standard to provide this documentation in a LLM readable format.

The following files are availlable:

- [`llms.txt`](/llms.txt)
- [`llms-full.txt`](/llms-full.txt)

Additionally, the documentation is availlable from the [Context7](https://context7.com/oktomusic/oktomusic) [MCP](https://modelcontextprotocol.io) server.

Refer to your agent documentation for more informations on how to get the best from these resources.
