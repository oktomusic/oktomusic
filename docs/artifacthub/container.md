# Oktomusic

Oktomusic is a open-source, self-hosted music streaming platform that allows you to stream your music collection with a modern and responsive web interface.

It is opinionated by design and provides the following features:

- 🌐 Web app with [**PWA (Progressive Web App)**](https://web.dev/explore/progressive-web-apps) support
- 🔒 Full reliance on a [**OpenID Connect**](https://openid.net/developers/how-connect-works) provider of your choice for authentication and roles assignement ([**Keycloak**](https://www.keycloak.org), [**Authentik**](https://goauthentik.io), [**Kanidm**](https://kanidm.com), etc)
- 📃 Syncronised lyrics support ([**TTML**](https://en.wikipedia.org/wiki/Timed_Text_Markup_Language), [**LRC**](<https://en.wikipedia.org/wiki/LRC_(file_format)>))
- 📤 Playlist exporting ([**XSPF**](https://xspf.org), [**M3U**](https://en.wikipedia.org/wiki/M3U))
- 🐳 [**Docker**](https://www.docker.com) first with secure and optimised official image
